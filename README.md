# archiveasite

preserve any webpage as a single html file

## how to use
it's really easy as long as you are above the age of around 3. (no hate to toddlers)

1. go to the website: https://archiveasite.vercel.app/
2. enter a url
3. press the big purple button

see? i knew you could do it

## why tho
- works completely offline after download
- no external dependencies
- one file = easy to share
- css and images all embedded

## tech stack
- next.js
- tailwind css
- cheerio for html parsing
- vercel for hosting
